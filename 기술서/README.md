## Mimir 한 줄 요약

> **Mimir = 범용 문서/지식 플랫폼 + API 기반 인프라**
>
> 모든 문서 유형을 Generic 구조로 수용하고, DocumentType 플러그인이 타입별 동작을 정의한다.
> API-first 설계로 UI와 외부 시스템 모두 동일한 API를 통해 플랫폼을 활용한다.
