## 2. Database

| 항목 | 기술 | 버전 | 선택 근거 |
|------|------|------|-----------|
| RDBMS | PostgreSQL | 16 | JSONB, GIN 인덱스, tsvector FTS, pgvector 확장 |
| 벡터 확장 | pgvector | - | embedding 저장, HNSW 인덱스, cosine 유사도 |
| FTS | PostgreSQL tsvector | - | 별도 검색 엔진 없이 전문 검색 구현 |
| 연결 풀 | ThreadedConnectionPool | - | 동기 FastAPI 엔드포인트에서 안전한 연결 재사용 |

### 핵심 PostgreSQL 기능 활용

```
JSONB         → metadata, plugin_config, schema_fields 확장 저장
tsvector/GIN  → documents, versions, nodes 전문 검색 (FTS)
pgvector/HNSW → document_chunks 임베딩 ANN 검색
UUID          → 모든 기본 키 (gen_random_uuid())
TIMESTAMPTZ   → 시간대 포함 타임스탬프 전 컬럼
```

## 6. 보안

| 항목 | 기술 | 역할 |
|------|------|------|
| 인증 | JWT HS256 | Access token (기본 120분) |
| API 인증 | SHA-256 해시 + HMAC | API Key 검증 (타이밍 공격 방지) |
| 세션 | PBKDF2 기반 토큰 | 쿠키 세션 |
| 보안 헤더 | SecurityHeadersMiddleware | HSTS, CSP, X-Frame-Options 등 7개 헤더 |
| 정적 분석 | Bandit | CI에서 HIGH 이상 취약점 검출 시 빌드 실패 |
| 린팅 | Ruff | Python 코드 품질 (E/W/F/I/B/S/UP) |
