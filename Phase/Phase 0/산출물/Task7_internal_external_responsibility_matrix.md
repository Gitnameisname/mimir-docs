# Phase 0 — 내부/외부 책임 매트릭스

## 사용 목적

이 매트릭스는 플랫폼의 각 기능 영역이 내부 코어 책임인지, 외부 시스템과 연계하는 영역인지를 구분한다. 기능 추가 요청이나 설계 방향 논쟁 시 이 매트릭스를 기준으로 판단한다.

### 분류 기준

| 분류 | 의미 |
|------|------|
| **코어** | 플랫폼이 직접 설계·구현·운영 책임. 외부 위임 불가. |
| **코어 + 연계** | 플랫폼이 핵심 기능을 담당하되, 외부 시스템과의 연동 인터페이스도 제공. |
| **연계 전담** | 외부 시스템이 기능 담당. 플랫폼은 참조·통합 인터페이스만 제공. |
| **후속 검토** | 현재 범위 밖. 필요성이 확인되면 후속 Phase에서 검토. |
