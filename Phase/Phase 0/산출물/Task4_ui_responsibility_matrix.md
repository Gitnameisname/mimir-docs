# Phase 0 — UI 책임 범위 매트릭스

## 사용 목적

이 매트릭스는 플랫폼의 각 기능·영역이 User UI, Admin UI, 또는 공통 영역에 해당하는지 판별하는 기준이다. 기능 추가나 화면 설계 시 해당 기능이 어느 UI에 속해야 하는지 판단하는 근거로 사용한다.

### 분류 기준

| 분류 | 의미 |
|------|------|
| **User** | User UI에서 주로 수행. 일반 사용자 권한으로 접근 가능. |
| **Admin** | Admin UI에서만 수행. 관리자 권한 필요. |
| **공통 (관점 분리)** | 같은 기능이지만 User와 Admin이 보는 정보와 허용 동작이 다름. |
| **User 노출 금지** | Admin 전용 기능. User UI에 어떤 형태로도 노출하지 않음. |
