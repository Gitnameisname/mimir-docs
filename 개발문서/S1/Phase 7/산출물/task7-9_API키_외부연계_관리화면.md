|------|--------|------|
| 키 이름/별칭 | `name` | |

| 마지막 사용 | `last_used_at` | 90일 미사용 시 경고 표시 |
| 발급일 | `created_at` | |

## 6. 보안 경고 및 마스킹 규칙 문서

| 상황 | 처리 방식 |
|------|----------|
| 키 전체 값 표시 | 발급 시 1회만 — 이후 항상 `****` 마스킹 |
| 만료 임박 (7일 이내) | 노란 배지 + 목록 상단 경고 배너 |
| 이미 만료된 키 | 회색 배지 + 재발급 안내 메시지 |
| 장기간 미사용 (90일+) | 주황 아이콘 + "미사용 경고" 툴팁 |
| 연계 설정값 (토큰/시크릿) | 항상 `[MASKED]` 표시 |
| REVOKED 키 | 회색 + 취소선, 폐기 사유 표시 |
