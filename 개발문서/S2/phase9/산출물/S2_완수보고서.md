## 1. 정의 및 성과

### 1.1 S2 정의

> **S2 = Mimir를 "AI 에이전트가 1급 시민으로 소비·기록·행동하는 지식 인프라"로 진화시키는 스프린트**

S1이 기초 문서 관리 플랫폼(검색, 버전 관리, 인증, 워크플로)을 구축했다면, S2는 그 위에 **AI 에이전트를 동등한 API 소비자**로 온보딩하고, 에이전트의 행동에 **안전장치(Draft 제안, Kill Switch, Scope ACL)**를 부여하며, LLM이 생성하는 모든 콘텐츠에 **검증 가능한 출처(Citation 5-tuple)**를 붙이는 데 집중했다.

## 2. 기술 성과

### 2.1 핵심 기능

**LLM 추상화 (Phase 0~1)**
- OpenAI (`gpt-4o-mini`, `gpt-4o`) ↔ Anthropic (`claude-sonnet-4-6`) 동적 전환
- 폐쇄망: `embedding_service_url` / `llm_base_url` 환경변수로 로컬 서버 연결
- NullReranker, FTSRetriever로 벡터/LLM 없이도 검색 기능 유지

**검색 및 Citation (Phase 2)**
- Hybrid Retriever (FTS + Vector) + Cross-Encoder Reranker
- Citation 5-tuple: `(document_id, version_id, node_id, content_hash, span_offset)`
- SHA-256 content hash로 인용 무결성 보장

**대화 및 RAG (Phase 3)**
- MultiturnRAGService: 대화 이력 기반 컨텍스트 압축 + 검색
- PromptBuilder: system / context / query / search 섹션 분리
- Content-Directive 격리: 검색 결과를 `[UNTRUSTED CONTENT]`로 마킹

**MCP Agent Interface (Phase 4)**
- MCP 2025-11-25 HTTP 서버: `initialize`, `tools/list`, `tools/call`
- 지원 도구: `search_documents`, `fetch_node`, `verify_citation`
- `_CURATED_TOOLS` allowlist + scope ACL (`apply_scope_filter`) + injection 탐지
- 에이전트별 rate limiting (분당 20회)

**Agent Action Plane (Phase 5)**
- Draft 제안 패턴: 에이전트 쓰기 → `proposed` 상태 → 인간 승인 후 `published`
- Kill Switch: async 구현, `is_active=False`로 즉시 에이전트 비활성화
- FG5.2 Prompt Injection 방어: 룰 기반 탐지(40개 패턴), ContentDirectiveSeparator

**관리자 기능 (Phase 6)**
- Admin UI: Scope Profile 관리, Agent 등록/비활성화, Kill Switch 콘솔
- `require_authenticated` / `require_role` default-deny 인증 정책

**AI 품질 평가 (Phase 7)**
- ExtractionEvaluator: field accuracy, span accuracy, hallucination rate
- GoldenSet import/export, CI 게이트 통합
- LLMOutputValidator: 스키마 검증 + HTML sanitizer
- PiiDetector: 이메일, 전화번호, 주민번호 등 탐지

**Structured Extraction (Phase 8)**
- ExtractionPipelineService: 문서 → 구조화 필드 자동 추출
- BatchExtraction: 대량 문서 비동기 배치 처리
- SourceSpan: 추출 필드 ↔ 원문 위치 역참조
- DiffCalculator: 추출 결과 버전 간 비교

### 2.2 보안

- **OWASP Top 10 (A01~A10)**: 전 항목 점검 완료, High/Critical 취약점 0건
- **OWASP LLM Top 10 (LLM01~LLM10)**: 전 항목 점검 완료
- **Prompt Injection 탐지**: 40개 패턴, 20/20 표준 공격 탐지
- **Kill Switch**: async 구현으로 5초 내 발동 보장
- **에이전트 쓰기**: 인간 승인 게이트 통과 없이 `published` 진입 불가
- **SSRF 방어**: 외부 URL 환경변수 전용 관리, 웹훅 https-only + 내부 IP 차단
- **의존성 스캔**: pip-audit CI 통합 (weekly)

- OWASP Top 10 + LLM Top 10 전 항목 통과
- S2 원칙 ⑤⑥⑦ 전면 적용 확인
- 448개 자동화 테스트 100% 통과
- CI/CD 통합 완료 (보안 + 회귀 + 품질 3종 워크플로우)

S3에서는 외부 커넥터 확장 및 AI-native 계획 도구 개발을 통해 Mimir의 지식 인프라 범위를 더욱 넓힐 수 있는 견고한 기반이 마련되었다.
