## 1. MCP 연결 가이드

### 1.1 기본 연결 설정

```python
import httpx

MCP_BASE_URL = "http://mimir-backend:8000/api/v1/mcp"
AGENT_API_KEY = "your-agent-api-key"  # 환경변수 권장

headers = {
    "X-API-Key": AGENT_API_KEY,
    "Content-Type": "application/json",
}
```

### 1.2 핸드셰이크 (initialize)

```python
def mcp_initialize(client: httpx.Client) -> dict:
    resp = client.post(
        f"{MCP_BASE_URL}/initialize",
        headers=headers,
        json={
            "protocolVersion": "2025-11-25",
            "clientInfo": {"name": "my-chatbot", "version": "1.0.0"},
        },
    )
    resp.raise_for_status()
    return resp.json()

# 반환값 구조:
# {

```

### 1.3 문서 검색 (search_documents)

```python
def search_documents(
    client: httpx.Client,
    query: str,
    scope: str = "public",
    access_context: dict | None = None,
    top_k: int = 5,
) -> list[dict]:
    resp = client.post(
        f"{MCP_BASE_URL}/tools/call",
        params={"tool_name": "search_documents"},
        headers=headers,
        json={
            "params": {
                "query": query,
                "scope": scope,
                "access_context": access_context or {},
                "top_k": top_k,
            }
        },
    )
    resp.raise_for_status()
    data = resp.json()
    if not data.get("ok"):
        raise RuntimeError(f"MCP error: {data.get('error')}")
    return data["result"]["results"]

# 반환값 구조 (각 result):
# {
#   "document_id": "uuid",
#   "title": "문서 제목",
#   "content": "청크 텍스트",
#   "citation": {
#     "document_id": "uuid",
#     "version_id": "uuid",
#     "node_id": "uuid",
#     "content_hash": "sha256-64자",
#     "span_offset": null
#   },
#   "score": 0.92
# }
```

### 1.4 노드 상세 조회 (fetch_node)

```python
def fetch_node(client: httpx.Client, node_id: str) -> dict:
    resp = client.post(
        f"{MCP_BASE_URL}/tools/call",

```

### 1.5 Citation 검증 (verify_citation)

```python
def verify_citation(
    client: httpx.Client,
    document_id: str,
    content_hash: str,
) -> bool:
    resp = client.post(
        f"{MCP_BASE_URL}/tools/call",
        params={"tool_name": "verify_citation"},
        headers=headers,
        json={"params": {"document_id": document_id, "content_hash": content_hash}},
    )
    resp.raise_for_status()
    return resp.json().get("result", {}).get("valid", False)
```

## 2. Scope Profile 매핑 가이드 (S2 원칙 ⑥)

### 2.1 Scope 종류

| scope 값 | 접근 범위 | 사용 시나리오 |
|----------|-----------|--------------|
| `"public"` | 공개 문서 | 외부 챗봇, 비인증 에이전트 |
| `"team"` | 팀 문서 (access_context 필요) | 팀 전용 지식베이스 |
| `"organization"` | 조직 전체 문서 | 내부 전사 지식 |
| `"private"` | 개인 문서 | 사용자 본인 문서만 |

### 2.2 access_context 구조

```python
access_context = {
    "user_id": "user-uuid",           # 에이전트 대리인 사용자
    "organization_id": "org-uuid",    # 조직 UUID
    "team_id": "team-uuid",           # 팀 UUID (scope="team"일 때 필수)
    "actor_type": "agent",            # S2 원칙 ⑤: "user" 또는 "agent"
}
```

## 4. MCP 성능 최적화

### 4.1 연결 재사용 (httpx.Client 재사용)

```python
# ❌ 비효율 — 요청마다 새 연결
for query in queries:
    resp = httpx.post(...)

# ✅ 효율 — 클라이언트 재사용
with httpx.Client(timeout=30.0) as client:
    for query in queries:
        results = search_documents(client, query)
```

### 4.2 병렬 요청 (asyncio)

```python
import asyncio
import httpx

async def search_all(queries: list[str]) -> list:
    async with httpx.AsyncClient(timeout=30.0) as client:
        tasks = [
            client.post(
                f"{MCP_BASE_URL}/tools/call",
                params={"tool_name": "search_documents"},
                headers=headers,
                json={"params": {"query": q, "scope": "public", "top_k": 3}},
            )
            for q in queries
        ]
        responses = await asyncio.gather(*tasks)
        return [r.json()["result"]["results"] for r in responses]
```

### 4.3 Rate Limit 준수

```
에이전트 rate limit: 분당 20회 tool_call
초과 시: HTTP 429 응답
```

```python
import time

def safe_search(client, query, retries=3):
    for attempt in range(retries):
        resp = client.post(...)
        if resp.status_code == 429:
            wait = 2 ** attempt  # 1s, 2s, 4s
            time.sleep(wait)
            continue
        resp.raise_for_status()
        return resp.json()
    raise RuntimeError("Rate limit exceeded after retries")
```

## 5. 에러 처리 및 재시도 전략

### 5.1 에러 코드 대응

| HTTP 상태 | MCP error code | 대응 |
|-----------|---------------|------|
| 401 | `UNAUTHORIZED` | API Key 확인, 에이전트 활성 상태 확인 |
| 403 | `FORBIDDEN` | Scope Profile 권한 확인 |
| 404 | `NOT_FOUND` | document_id / node_id 존재 여부 확인 |
| 429 | `RATE_LIMITED` | Exponential backoff 후 재시도 |
| 400 | `INVALID_REQUEST` | 파라미터 유효성 확인 |
| 500 | `INTERNAL_ERROR` | Mimir 서버 로그 확인, 재시도 1회 |

### 5.2 재시도 권장 패턴

```python
def with_retry(fn, retries=3, base_delay=1.0):
    last_exc = None
    for attempt in range(retries):
        try:
            return fn()
        except httpx.HTTPStatusError as e:
            if e.response.status_code in (429, 503):
                time.sleep(base_delay * (2 ** attempt))
                last_exc = e
            else:
                raise  # 재시도 불가 에러
    raise last_exc
```

## 8. 평가 지표 해석 (Phase 7)

### 8.1 RAG 품질 지표

| 지표 | 임계값 | 의미 | 낮을 때 조치 |
|------|--------|------|-------------|
| Faithfulness | ≥ 0.80 | 답변이 출처와 일치하는 정도 | 시스템 프롬프트에 "출처 외 내용 금지" 강화 |
| Citation-present rate | ≥ 0.90 | 답변 문장 중 Citation 있는 비율 | Prompt에 "[숫자] 형식 인용 필수" 명시 |
| Answer Relevance | ≥ 0.75 | 쿼리와 답변의 관련성 | top_k 증가, 쿼리 재작성 활성화 |
| Hallucination rate | ≤ 0.10 | 출처 없는 내용 생성 비율 | 온도(temperature) 낮추기 (≤ 0.3) |

### 8.2 지표 조회 API

```python
# 특정 기간 RAG 품질 지표 조회
resp = httpx.get(
    f"{MIMIR_BASE}/api/v1/admin/evaluations/summary",
    params={"from": "2026-04-01", "to": "2026-04-18"},
    headers=admin_headers,
)
metrics = resp.json()["data"]["aggregate"]
print(f"Faithfulness: {metrics['faithfulness_mean']:.3f}")
print(f"Citation rate: {metrics['citation_present_rate']:.3f}")
```
