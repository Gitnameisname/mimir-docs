### 4-2. 스트리밍 응답 컴포넌트

SSE 스트리밍을 구독하여 실시간 렌더링:

```
- 응답 텍스트가 청크 단위로 도착하며 화면에 순차 추가
- 스트리밍 중 로딩 커서(▋) 표시
- 스트리밍 완료 후 Citations 섹션 표시
- 마크다운 렌더링 지원
```

#### SSE 구독 로직

```typescript
const eventSource = new EventSource('/api/rag/query', { method: 'POST', ... });

eventSource.onmessage = (event) => {
  const data = JSON.parse(event.data);
  if (data.type === 'chunk') {
    appendToAnswer(data.content);
  } else if (data.type === 'citations') {
    setCitations(data.citations);
  } else if (data.type === 'done') {
    setComplete(data.conversation_id, data.message_id);
    eventSource.close();
  } else if (data.type === 'error') {
    setError(data.message);
    eventSource.close();
  }
};
```

### 4-3. Citation 표시 컴포넌트

응답 내 `[1]`, `[2]` 마킹 클릭 시 원본 문서 이동:

```
응답 텍스트:
  "접근통제는 역할 기반으로 처리됩니다. [1]
   최소 권한 원칙이 적용됩니다. [2]"

출처 섹션:
  [1] 정보보안 기본 정책 > 2.3 접근통제 → [문서 보기]
  [2] 정보보안 기본 정책 > 2.4 최소 권한 → [문서 보기]
```

- `[1]` 클릭 → 해당 문서 노드로 이동 (새 탭 또는 슬라이드 패널)
- 문서 보기 링크: `/documents/{document_id}?node={node_id}`

### 4-5. 문서 상세 페이지 RAG 패널 (선택)

문서 상세 페이지 우측에 RAG 패널을 추가하여 현재 보고 있는 문서에 대해 바로 질문:

```
[문서 내용]        [이 문서에 대해 질문하기]
                   Q: 접근통제 기준이 뭔가요?
                   A: 접근통제는... [1]
```

document_type 필터를 자동으로 현재 문서의 타입으로 설정.
