## 2. 설계 판단

### 2.1 블로커1 결정서 § 3.2 "서버 파서가 정본"

클라이언트의 TipTap `HashtagMark` (프런트 추후 이식) 는 **시각 강조용**이며, 태그 인덱스의 **정본은 서버 파서** (`extract_tags_from_snapshot`). 이 규약으로:
- frontmatter tag 와 인라인 tag 를 한 곳에서 **union** 처리 → `source ∈ {inline, frontmatter, both}`
- 클라이언트가 mark 를 넣지 않아도 서버가 재파싱해 정규화 (Vault Import 경로 호환)

### 2.2 정규화 규칙 (블로커1 결정서 §3.3)

```
normalize_tag:
  1. strip
  2. 앞의 '#' 1개까지 허용 (인라인 토큰)
  3. NFKC → lowercase
  4. pattern [\w/-]{1,64} fullmatch
  5. 실패 시 None (필터/수집 스킵)
```

- **대소문자/전각 흡수**: `#AI` / `ＡＩ` / `  ai  ` → `ai`
- **길이 상한 64**: DB `VARCHAR(64) UNIQUE` 와 일치
- **중첩 태그**: `ai/ml` 허용
- **`##heading` 방어**: 앞에 `#` 두 개 이상이면 reject (Markdown heading 혼동 방지)

### 2.3 정규식 `INLINE_HASHTAG_PATTERN`

```
(?:^|(?<=\s)|(?<=[^\w#]))
#
([\w][\w/-]{0,63})
(?=\s|$|[^\w/-])
```

- 앞 경계: 줄 시작 / 공백 뒤 / 단어·해시 아닌 문자 뒤
- 뒤 경계: 공백 / 끝 / 허용 외 문자 (단어 경계)
- **코드블록 노드 제외** — text run 재귀 순회 시 `type="codeBlock"` 스킵
- **link mark text 제외** — `marks` 에 `{type: "link"}` 가 있으면 텍스트 자체 스킵

### 2.4 파생 동기화 트랜잭션 규약

작업지시서 상호검증 §7 에 확정된 순서:

```python
# 쓰기 경로 (save_draft / agent 승인)
with get_db() as conn:
    rebuild_nodes_from_snapshot(conn, version_id, content_snapshot)   # FG 1-1 (Phase 1)
    rebuild_tags_for_document(conn, document_id, content_snapshot, metadata)  # FG 2-2 (이번)
    # rebuild_wikilinks_for_document(...) — FG 2-3 후속
```

세 함수가 **동일 트랜잭션**에서 실행. 예외 시 전량 롤백 → 정본 이전 상태 유지.

### 2.5 ACL — 태그는 Scope 영향 없음

- `tags` 전역 풀 (Scope 무관)
- `document_tags.tag_id IN (...)` 은 scope 필터와 **독립**
- "태그 → 문서" 방향 (`document_ids_for_tag`) 에서는 **documents INNER JOIN** + `d.scope_profile_id IN (...)` 로 자동 필터
- `/documents?tag=<name>` 도 동일 — 기존 `_build_list_query` 의 scope 필터와 AND 결합

k-anonymity (블로커1 §5): `popular` 에 `min_usage=1` 기본값 — 실운영에서 3 이상으로 올릴 수 있음.

### 2.6 서버 측 입력 정규화

`/documents?tag=AI` 로 들어오면 `_build_list_query` 가 `normalize_tag("AI") → "ai"` 로 변환 후 쿼리에 사용. 클라이언트가 어떤 표기로 보내든 같은 결과. `/tags?q=AI` 도 라우터에서 동일 처리.

| 메서드 | 경로 | 설명 | 권한 |
|-------|------|------|------|
| GET | `/api/v1/tags?q=<prefix>&limit=` | 자동완성 (prefix + usage 순) | 인증된 모두 |
