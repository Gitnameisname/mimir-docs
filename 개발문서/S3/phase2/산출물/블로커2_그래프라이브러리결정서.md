## 1. 배경

FG 2-4 는 문서를 노드, 백링크/컬렉션/태그를 엣지/메타노드로 하는 **문서 그래프 뷰** + 4종 레이아웃(리스트/트리/카드/그래프) 을 제공한다. 폐쇄망 환경에서 번들로 동작해야 하며, CLAUDE.md 전역 §2 에 따라 **critical 취약점이 있는 라이브러리 사용 금지**.

## 2. 후보 비교

### 2.1 sigma.js (v3)

| 항목 | 평가 |
|------|-----|
| 라이선스 | MIT |
| 렌더러 | WebGL + Canvas fallback |
| 강점 | 대규모(수만 노드) 렌더링, 하드웨어 가속 |
| 약점 | **레이아웃 알고리즘 내장이 빈약** — `graphology-layout` 계열 별도 설치 필요. `ForceAtlas2`, `noverlap`, `circular` 등. 트리/계층 레이아웃은 약함 |
| 타입스크립트 | 공식 지원 (`@types` 번들) |
| 생태계 | 중소 규모. 최근 활동 양호 |
| 번들 크기 | ~ 150 KB (gzip, sigma + graphology 합계) |
| 보안 이력 | 최근 critical CVE 없음 (2026-04 시점 npm audit 기준 예상, 실제 설치 시 재검증 필요) |

### 2.2 cytoscape.js (v3) ✅

| 항목 | 평가 |
|------|-----|
| 라이선스 | MIT |
| 렌더러 | Canvas + optional WebGL (v3.28+) |
| 강점 | **레이아웃 다양성** — breadthfirst / concentric / circle / grid / random / cose / cose-bilkent (plugin) / cola (plugin) / dagre (plugin) / euler (plugin) / klay (plugin). 트리, 계층, 힘기반, 원형 모두 지원 |
| 약점 | 수만 노드 이상에서는 sigma.js 대비 느림 — 단, Phase 2 타겟 규모(문서 수백~수천, 엣지 수천~수만) 에서 문제 없음 |
| 타입스크립트 | 공식 지원 (`@types/cytoscape`) |
| 생태계 | 큼. 학술/산업 양쪽에서 널리 쓰임. 생물학 네트워크 시각화에서 유래 |
| 번들 크기 | ~ 210 KB (gzip, core) + 각 레이아웃 플러그인 ~ 20~40 KB |
| 보안 이력 | 최근 critical CVE 없음. 성숙한 보안 대응 히스토리. 실 설치 시 `npm audit --production` 재확인 |

### 2.3 기타 후보 (기각)

- **vis-network**: MIT. 사용자 감소, 유지보수 활동 약화. 공식 문서가 뒤처짐. → 기각
- **react-flow**: MIT-ish 라이선스 (최근 상업 라이선스 조항 논란). 그래프 시각화보다는 워크플로우 편집기 성격. → 기각
- **d3-force 직접 구현**: 커스텀 부담 큼. Phase 2 스코프에서 자체 엔진까지 만들 시간 없음. → 기각

## 3. 결정 근거

1. **레이아웃 4종 전환 요구를 단일 라이브러리로 해결.** cytoscape.js 의 breadthfirst / concentric / cose-bilkent / dagre 만으로 "리스트-like 선형 / 동심원 / 힘기반 / DAG" 를 전부 커버.
2. **Mimir 의 그래프 규모는 수천~수만 엣지 수준** — cytoscape.js 성능 한계보다 훨씬 아래. sigma.js 의 WebGL 가속은 오버스펙.
3. **폐쇄망 번들 확인**: 두 라이브러리 모두 단일 `npm install` + Webpack/Vite 번들로 오프라인 동작. 외부 CDN 의존 없음. cytoscape.js 가 이미 오프라인 엔터프라이즈 시장에서 검증된 케이스가 많음.
4. **레이아웃 플러그인 격리**: 각 레이아웃은 별도 패키지로 분리되어, FG 2-4 가 레이아웃 4종만 탑재하고 나머지는 트리쉐이킹으로 제외 가능.
5. **CLAUDE.md 전역 §2 "critical 취약점 있으면 사용 금지"** — 설치 시점 `npm audit --production --audit-level=critical` 로 점검. 현재까지 cytoscape.js / 공식 플러그인 critical 이력 없음 (2026-04 시점).
6. **TypeScript 호환성** — Mimir 는 Next.js 16 + TypeScript 프로젝트. cytoscape 는 `@types/cytoscape` 공식 제공.

## 4. 설치 / 번들 계획

### 4.1 npm 의존성

```json
{
  "cytoscape":                "^3.30.0",
  "cytoscape-cose-bilkent":   "^4.1.0",
  "cytoscape-dagre":          "^2.5.0",
  "@types/cytoscape":         "^3.21.0"
}
```

- `cytoscape-dagre` 는 `dagre` 를 peerDependency 로 가짐 → `dagre@^0.8.5` 를 같이 설치
- 번들 크기 예상: core 210 KB + cose-bilkent 30 KB + dagre 연관 ~ 70 KB (gzip)

### 4.2 폐쇄망 검증 절차 (FG 2-4 구현 시 필수)

1. `npm install --prefer-offline` 이후 로컬 캐시로만 의존성 해결되는지 확인
2. 프로덕션 빌드 (`next build`) 후 `.next/static/chunks/` 내 cytoscape 청크가 단일 asset 으로 묶여 있고 외부 fetch 호출 없는지 확인
3. 네트워크 차단 상태에서 그래프 뷰 열기 → 렌더 성공 확인
4. CSP 헤더와 cytoscape 의 Canvas 사용이 충돌 없는지 확인 (`img-src`, `style-src` 완화 불필요)

### 4.3 보안 검증 절차

1. `npm audit --production --audit-level=critical` exit 0 확인
2. `npm ls cytoscape dagre cytoscape-dagre cytoscape-cose-bilkent` 로 하위 트리 전체 취약점 재점검
3. 취약점 발견 시 **FG 2-4 진행 중단 → 대체 라이브러리 재평가** (sigma.js 로 폴백)

## 5. FG 2-4 구현 개요 (본 결정 기준)

- **레이아웃 4종 매핑**:
  - 리스트 = TanStack Table / 그래프 없음 (비 cytoscape)
  - 트리 = cytoscape + `breadthfirst` 레이아웃 (폴더/컬렉션 기준 부모 → 자식)
  - 카드 = 카드 그리드 (비 cytoscape, CSS Grid)
  - 그래프 = cytoscape + `cose-bilkent` 기본 / `dagre` 토글 가능
- 토글 UI + 키보드 단축키 (`⌘/Ctrl+Alt+1/2/3/4`)
- 노드 상한 초기값 500, "더 불러오기" 버튼. 뷰포트 컬링은 cytoscape 기본 동작 활용
- 클릭 이벤트: 문서 노드 → `/documents/{id}` 이동, 태그 노드 → `?tag=<name>` 필터

## 6. 리스크 / 대응

| 리스크 | 대응 |
|--------|-----|
| 수천 노드 이상에서 레이아웃 계산 지연 | `layout.stop()` API 로 5 초 제한 + 워커 실행 (`cose-bilkent` 는 worker 옵션 지원) |
| 번들 크기 증가로 초기 로딩 지연 | dynamic import (`next/dynamic`) 로 그래프 페이지에서만 로드 |
| cytoscape CSS 가 전역 오염 | `<div>` 루트에 범위 클래스 부여, cytoscape 가 내부적으로 canvas 만 사용하므로 실제 CSS 오염 없음 |
| cytoscape-dagre의 peerDependency dagre 의 CVE | 설치 시점 + 분기 실측. 발견 시 dagre 고정 버전 + 패치 적용 |

## 7. 기각된 sigma.js 로의 재검토 조건

다음 조건 중 하나가 성립하면 **sigma.js 로 재평가**:

- cytoscape.js 최신 메이저에 critical CVE 발생 + 패치 지연
- 실 운영에서 그래프 노드가 **5,000개 이상** 로 지속 증가 (백링크 폭발)
- cytoscape.js 프로젝트가 유지보수 중단 (1 년 이상 릴리스 없음)
