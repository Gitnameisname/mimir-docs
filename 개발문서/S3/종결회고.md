- **Phase 0**: FG 0-1 / 0-2 / 0-4 / 0-5 는 *선언적 종결*. 운영자 후속 (CI 녹색 3회, 실 LLM/임베딩 측정, Alembic 적용 검증) 미수행 — Phase1_인수문서.md §1 참조. 따라서 release/readiness 산출물로 사용 시 본 항목들의 운영자 실측이 선행되어야 한다.
- **FG 3-3 게이트 46 (MCP 권한 게이트)**: *부분 통과*. tool-level allow 체크 (`allowed_tools` / `can_read`) 미구현, §4.1 #11 으로 등재.
- **FG 3-3 게이트 51 (node:test ≥ 30)**: *⚠️ 11/30* — TipTap mark/gutter 미도입에 따른 결정 사유 (FG3-3 종결보고서 §3) 명시.
- **FG 3-3 게이트 47 (자동완성)**: 별 라운드 deferred — Phase 5 흡수.
- **잔존 skip 124~126**: Phase 0 베이스라인 잔존, S3 신규 도입은 0건.

Phase 3 종결 시점의 **156 백엔드 unit + 551 프런트엔드 unit + tsc 0 error** 가 베이스라인을 보존했다.
